# PaidClinic - Учет платной клиники

## Описание
Приложение для учета пациентов, врачей и обращений в платной клинике.

## Функциональность
- Управление врачами (добавление, редактирование, удаление)
- Управление пациентами
- Учет обращений (приемов)
- Поиск и фильтрация
- Экспорт данных

## Технологии
- Avalonia UI (.NET 8)
- MVVM паттерн
- ReactiveUI
- Access/SQLite база данных

## Установка
1. Убедитесь что установлен [.NET 8 SDK](https://dotnet.microsoft.com/download/dotnet/8.0)
2. Клонируйте репозиторий:
```bash
git clone https://github.com/ваш-username/PaidClinic.git=
```
